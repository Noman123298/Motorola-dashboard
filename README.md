## 📝 Project Overview
This repository contains my **first-ever Power BI Dashboard**, focused on analyzing sales performance for a multi-brand mobile retail dataset. The goal of this project was to transform raw transactional data into actionable business insights, covering sales trends, customer behavior, and regional performance.

### 🎯 Key Objectives:
* Track overall revenue and transaction volume.
* Identify top-performing brands and mobile models.
* Analyze customer demographics and payment preferences.
* Visualize geographical sales distribution across India.

---

## 📊 Dashboard Highlights
The dashboard provides a comprehensive view of the retail landscape using several key metrics (KPIs) and visualizations:

* **Total Sales:** ₹769M | **Total Units Sold:** 19K | **Total Transactions:** 4K
* **Geospatial Analysis:** A map view showing unit distribution across major Indian cities (Delhi, Mumbai, Bangalore, etc.).
* **Brand Comparison:** Sales breakdown for top brands including **Apple, Samsung, OnePlus, Vivo, and Xiaomi**.
* **Payment Trends:** Analysis of transaction methods (UPI, Credit/Debit Cards, and Cash).
* **Customer Feedback:** Distribution of star ratings (1-5) to gauge product satisfaction.
* **Temporal Trends:** Daily and weekly sales fluctuations to identify peak shopping periods.

---

## 💾 About the Dataset
The underlying dataset consists of over **3,800+ rows** of transactional data. Key features include:
| Column | Description |
| :--- | :--- |
| **Transaction ID** | Unique identifier for each sale. |
| **Brand & Model** | Mobile manufacturer and specific model (e.g., iPhone 11, 12, SE). |
| **Units Sold** | Number of devices purchased per transaction. |
| **Price Per Unit** | The individual cost of the mobile device. |
| **Customer Info** | Name, Age, and Location (City). |
| **Payment Method** | UPI, Cash, Credit Card, or Debit Card. |
| **Rating** | Customer satisfaction score. |

## 🛠️ Tools & Skills Used
* **Power BI Desktop:** For data modeling and visualization.
* **Power Query:** Data cleaning and transformation (ETL).
* **DAX (Data Analysis Expressions):** Created measures for Total Sales, Average Order Value, and Ratings.
* **Data Visualization:** Custom themes, map integration, and interactive filtering.

## 🚀 How to Use
1.  Download the `.pbix` file from this repository.
2.  Open it using **Power BI Desktop**.
3.  Use the **Month Slicer** on the left to filter data for specific timeframes.
4.  Interact with the **Brand** and **Map** visuals to cross-filter the entire report.

---

## 💡 Key Insights Derived
* **UPI** is a leading payment method, accounting for over 26% of transactions.
* Sales peak significantly on specific days of the month, suggesting the impact of promotional events.
* Despite being a multi-brand store, premium brands like **Apple** maintain high total sales value due to higher price-per-unit.
